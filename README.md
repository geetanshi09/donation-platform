# Donation Platform 💝

A Spring Boot-based web application that allows users to create and manage donation campaigns.  
Donors can view active campaigns and contribute, while organizers can track progress toward their goals.  

## Features 🚀
- Create, view, and delete donation campaigns
- Track goal amount vs collected amount
- Manage campaigns via user-friendly web interface (Thymeleaf)
- MySQL database integration with JPA/Hibernate
- MVC architecture (Controller, Service, Repository, Entity layers)

## Tech Stack ⚙️
- **Backend:** Spring Boot, Spring MVC, Spring Data JPA
- **Frontend:** Thymeleaf, HTML, CSS
- **Database:** MySQL
- **Build Tool:** Maven
- **Version Control:** Git/GitHub
